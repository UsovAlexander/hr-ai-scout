Возможности

Что умеет HR-AI Scout

📡

Автоматический парсинг

Ежедневно собирает свежие резюме и вакансии с агрегаторов рынка труда без участия человека. Умная остановка по пустой странице предотвращает лишние запросы.

🔢

Числовой ID резюме

Декодирует числовой идентификатор резюме из hex-хеша URL (bytes 4–8), обеспечивая стабильную дедупликацию в ClickHouse.

📊

15 инженерных признаков

TF-IDF косинусная схожесть, совпадение локации, подсчёт скиллов в тексте вакансии, стаж в месяцах, зарплатные ожидания и другие.

⏱️

Расписание Airflow

Три DAG-а: ежедневный лёгкий сбор (5 стр.), еженедельное полное обновление (250 стр.), сбор вакансий по триггеру после резюме.

🔒

JWT-авторизация

OAuth2 Bearer-токены, роли пользователей (public / admin). История предсказаний в async SQLite с доступом только для администратора.

📈

MLflow-эксперименты

Все обучения и метрики логируются в MLflow. Лучшая модель — Logistic Regression с Optuna-гиперпараметрами.

Airflow

Расписание сбора данных

Три DAG-а работают последовательно — резюме всегда парсятся первыми, затем автоматически стартуют вакансии.

resume_daily

Ежедневный сбор резюме

Собирает первые 5 страниц (100 резюме) по каждой из 50 IT-профессий. Получает самые свежие анкеты.

🕐 пн–сб в 01:00 МСК · 5 стр. × 20 резюме
↓ триггерит vacancy_daily
resume_weekly

Еженедельное обновление

Ночь с субботы на воскресенье — полный пересбор базы. До 250 страниц на профессию, пропускает уже загруженные.

📅 сб в 23:00 МСК · 250 стр. × 20 резюме
↓ триггерит vacancy_daily
vacancy_daily

Сбор вакансий

Запускается только по триггеру от resume DAG-ов — никогда сам по себе. Гарантирует последовательность запросов к источникам данных.

🔗 по триггеру · 5 стр. × 20 вакансий
✓ schedule = None
Установка

Развернуть за 4 шага

1

Клонировать и установить зависимости

Python 3.9+, виртуальное окружение.

git clone https://github.com/UsovAlexander/hr-ai-scout.git
cd hr-ai-scout
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
2

Запустить ClickHouse

Хранилище резюме и вакансий. Создайте таблицы после запуска.

docker run -d --name clickhouse \
  -p 9000:9000 -p 8123:8123 \
  -e CLICKHOUSE_PASSWORD=pass \
  clickhouse/clickhouse-server
3

Поднять Airflow

Инициализация БД и создание admin-пользователя — автоматически.

cd airflow
docker compose up airflow-init
docker compose up -d
4

Запустить FastAPI

Сервис скоринга. Swagger UI на /docs.

cd app
uvicorn main:app --reload
# http://127.0.0.1:8000/docs
